## 🌟 Features

- **4 Cognitive Modes**: OOA (Experience), OODA (Reasoning), OOCA (Creative), OOHA (Discovery)
- **Auto Mode Switching**: AI selects best thinking approach automatically
- **Metacognitive Monitoring**: Self-assessment and confidence scoring
- **Usage Analytics**: Complete tracking and effectiveness reports
- **100% Test Coverage**: Production-ready quality

## 🚀 Quick Start

### Installation

```bash
# Install from ClawHub
clawhub install cognitive-flexibility

# Or clone from GitHub
git clone https://github.com/your-username/cognitive-flexibility.git
cd cognitive-flexibility
```

### Basic Usage

```python
from scripts.cognitive_controller import CognitiveController

# Create controller
controller = CognitiveController(confidence_threshold=0.7)

# Execute task (automatic mode selection)
task = "Analyze user feedback data and identify key issues"
result = await controller.process(task, tools=tools)

# View result
print(f"Mode: {result['mode']}")
print(f"Answer: {result['answer']}")
print(f"Confidence: {result['assessment']['overall_score']:.2f}")
```

## 🧠 Four Cognitive Modes

| Mode | Name | Best For | Driver |
|------|------|----------|--------|
| **OOA** | Experience Mode | Familiar scenarios | Memory-driven |
| **OODA** | Reasoning Mode | Complex analysis | Knowledge-driven |
| **OOCA** | Creative Mode | Innovation | Association-driven |
| **OOHA** | Discovery Mode | Research | Hypothesis-driven |
